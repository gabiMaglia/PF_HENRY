//////USER ENDPOINTS
Get all users       "localhost:3001/users/"      Trae todos los usuaiors sin direccion ni credenciales ni roles
Get user by id      "localhost:3001/users/:id"   Trae un usuaio completo
Post user           "localhost:3001/users/"      Crea usuario nuevo
Put user            "localhost:3001/users/:id"   Modifica usuario por id
Delete user by id   "localhost:3001/users/:id"   Elimina usuario por id
/////USER_ROLES ENDPOINTS
Get all Roles       "localhost:3001/user_role"          Trae todos los Roles disponibles 
Create new Role     "localhost:3001/user_role/create"   Edita Role de un usuario  / expects { role_name : "roleName"}
/////USER_CREDENTIALS ENDPOINTS
Get user cred byid  "localhost:3001/user_credentials/:id"  Trae las credenciales de un usuario
Edit user cred byid "localhost:3001/user_credentials/:id   Edita las credenciales de un usuario / expects { username : "username", password : "password"}
